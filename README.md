* **Objetivo:** Resolver os enigmas, para desbloquear os números para abrir a porta final
* **Controlos:**
    * **W, A, S, D:** Movimentar a personagem.
    * **Rato:** Olhar em redor.
    * **E / Clique:** Interagir com objetos.
    * **R :* Recarregar a Lanterna* 

## Como abrir o projeto
1. Instale o **Unity Hub** e a versão do Unity referida acima.
2. Faça o download ou clone este repositório do GitHub.
3. No Unity Hub, clique em **Add** -> **Add project from disk**.
4. Selecione a pasta principal deste projeto.
5. Após abrir, localize a pasta `Assets` e abra a cena `MenuPrincipal`.
6. Pressione o botão **Play** no topo do editor para correr o jogo.
